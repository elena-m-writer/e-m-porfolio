Code First: Girls

At the end of 2020, after finishing my MA, I decided to boost my skills by taking on something I had never done before - coding. I completed Code First: Girls' Introduction to Web Development course and was highly commended on my final project, creating the majority of a website aimed at inspiring young people isolated during the COVID-19 Pandemic. You can check out the Iso-Mate website here and have a look at the code on github here. The site was created in November 2020 and all information on the site is relevant to that time period.

On this course I learnt the basics of HTML and CSS as well as some JavaScript essentials. I have since created this website to keep using my skills and have a simple, flexbox based, responsive site for sharing what I get up to.

Typography

Since completing the Code First: Girls course I have become increasingly interested in designing for the web, specifically how we can use text to engage readers and create a certain vibe for a website. Looking at text formatting and design is valuable not only for web desinging but also editing, as it shows how the elements of a font can impact the effectiveness of written copy.

I am currently completing an online course in Typography for Designers which includes modules on Social Media Ad Design and Responsive Typography for Developers.
